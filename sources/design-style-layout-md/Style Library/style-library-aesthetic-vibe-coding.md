# Style Library for Non-Generic AI Vibe Coding

## Purpose
This file is a prompt-ready style vocabulary for website generation.  
Goal: reduce generic AI design output and increase clear, distinctive art direction while keeping creative flexibility.

## Global Rules
- Pick one primary style and optionally one secondary style.
- Keep body typography readable even when display typography is expressive.
- Keep layout behavior intentional: clear hierarchy, varied rhythm, and section purpose.
- Use style keywords in prompts exactly as written in each style entry.

## 1. Neoclassical
**Description:** Revival of Greco-Roman order, symmetry, and refined authority in modern digital form.  
**Mood:** Formal, timeless, noble, authoritative.  
**Image/photo/illustration style:** Architectural photography, marble sculpture references, balanced symmetrical framing, soft natural light.  
**Typography and font pairing:** Display serif (`Cinzel`, `Cormorant SC`) + readable classic serif (`EB Garamond`, `Libre Baskerville`) or restrained sans (`Inter`).  
**Color palette and color system options:** Alabaster, slate, olive gray, antique gold; optional monochrome variant with gold accents only.  
**Website layout styles:** Centered symmetry, strict columns, generous margins, ceremonial section pacing.  
**Occasion and AI trigger keywords:** luxury heritage, museum, legal, academia, institution, classical branding, neoclassical, greco-roman, marble, laurel, symmetry, timeless authority.
