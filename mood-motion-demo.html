Mood → Motion demo

Motion has mood the way color does: speed, easing, rhythm, direction, scale, opacity, blur, and color behavior combine into character. This demo reads mood-to-motion-map.json (feeling tags, merged motion personality tags, and library stacks). One cluster drives four preview planes — element, effect, interaction, and background — timed from React Bits, GSAP, Anime.js (text, SVG), Magic UI, Kokonut UI, and Animate UI primitives (patterns here are plain CSS for a static page).

Mood

Exactly 24 mood chips (phrases from your vocabulary, each mapped to one canonical feeling id). Pick a chip to drive the four planes; the dropdown still picks the JSON cluster for paste snippets. Motion look follows the chip’s feeling.
