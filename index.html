打不开页面或出现 ERR_EMPTY_RESPONSE？
通常是本机没有在 127.0.0.1:8765 跑 Python 服务，或端口被占用。
做法：双击 launch.command（需保持终端不关）或双击 open-standalone.command 用 file:// 打开 standalone.html。
动效代码区空白：请将 standalone.html 与同目录的 motion-snippets.bundle.js 一起拷贝；需要更新时在终端执行 node fetch-reactbits.mjs、node fetch-motion-docs.mjs 与 node build-catalog.mjs。
