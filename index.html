Vibe Prompt Generator

Pick options → generate → copy into your vibe coding workflow. Output capped at 4999 characters. Prompts use layers L0–L9 (execution order, layout, space, mood→concrete CSS, type, color, motion, governance, assets) plus [STYLE_LIB] from in-repo designer markdown. Prompts cite awesome-design-md / getdesign.md references plus agency-quality guardrails. Optional DESIGN.md deliverable for coding agents (YAML tokens + rationale).

Blank page or ERR_EMPTY_RESPONSE on localhost?
Usually no server is running on 127.0.0.1:8765, or the port is taken.
Fix: double-click launch.command (keep the terminal open) or double-click open-standalone.command to open standalone.html via file://.
Motion code panel empty: keep standalone.html in the same folder as motion-snippets.bundle.js. To refresh snippets, run node fetch-reactbits.mjs, node fetch-motion-docs.mjs, then node build-catalog.mjs (or npm run build).
